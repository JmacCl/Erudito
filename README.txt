External sources used: Jquery, Bootstrap, Feather

Setting up this project on your local machine: 

git clone this repository to get all files on your local machine. 
Using the environment with packages specified in requirements.txt, run migration commands python manage.py makemigrations eruditoapp and python manage.py migrate to get the database set up.
Finally, run python populate_erudito.py to populate the database with some example data.
